## 1. Objective
To automate the discovery and verification of "Trending" AI tools. Instead of hype, we provide truth. If a tool fails to install in a standard environment, we bury it. If it survives, we praise it.

## 2. The Workflow (The Autopsy)

### Phase 1: The Patrol (Fetch)
- **Source**: GitHub Trending (Python/AI topics).
- **Criteria**: 
    - Created < 30 days (Fresh).
    - Stars > 100 (Hype).
    - Has `requirements.txt` (Basic hygiene).

### Phase 2: The Chamber (Install Test)
- **Action**: 
    1. Clone repo to a temporary sandbox.
    2. Create a fresh virtual environment.
    3. Run `pip install -r requirements.txt`.
